      \  Transformer le navigateur web en station de traitement audio /
      \_______________________________________________________________/

This is a 1st year final application student project at Telecom Paristech.

It consists in a lightweight browser application able to do simple audio processing like :
  - Recording and playing audio
  - Writing and recording .wav files
  - Moving and editing audio tracks
  - Adding basic effects : fade in, fade out, phaser, flanger, ...

Link to the full description in French : http://paf.telecom-paristech.fr/projets/transformer-navigateur-web-en-station-travail-audio
